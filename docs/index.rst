Running Igor - OSX
==================
1. Activate virtual environment
    source venv/bin/activate
2. Start skypekit (In a new tab)
    python /usr/local/skypekit/startskypekit.py --skypekit=./bin/mac-x86/mac-x86-skypekit
3. Start Igor
    python igor.py
4. Profit!
